# 正缘方位网 - 文章管理系统使用说明

## 一、系统概述

本系统提供了一个简单易用的文章管理功能，允许管理员通过浏览器在线编辑、添加、删除文章，无需复杂的后台系统。

## 二、文件结构

```
├── index.html              # 主网站文件
├── articles.json           # 文章数据文件
├── admin.html              # 文章管理页面
└── README.md              # 本使用说明文档
```

## 三、使用方法

### 1. 访问管理页面

在浏览器中输入以下地址访问文章管理页面：

```
https://www.fatenet.cn/admin.html
```

### 2. 登录

使用以下默认管理员密码登录：

```
密码：admin123
```

**注意：** 建议首次登录后修改密码，可在 `admin.html` 文件中修改 `ADMIN_PASSWORD` 常量。

### 3. 文章管理

#### 3.1 查看文章列表

登录后，默认进入"文章列表"页面，显示所有文章。

- 可以通过"分类筛选"下拉菜单筛选不同分类的文章
- 文章按发布日期倒序排列

#### 3.2 添加新文章

1. 点击"添加文章"标签页
2. 填写文章信息：
   - 标题：文章标题
   - 摘要：文章简短描述
   - 内容：文章完整内容
   - 分类：选择文章分类
   - 发布日期：选择发布日期
   - 点赞数：初始点赞数
3. 点击"保存文章"按钮
4. 文章将自动添加到文章列表

#### 3.3 编辑文章

1. 在文章列表中找到要编辑的文章
2. 点击文章下方的"编辑"按钮
3. 修改文章信息
4. 点击"保存文章"按钮

#### 3.4 删除文章

1. 在文章列表中找到要删除的文章
2. 点击文章下方的"删除"按钮
3. 在弹出的确认框中点击"确定"

### 4. 数据备份与恢复

#### 4.1 导出文章数据

1. 点击"导入导出"标签页
2. 点击"导出为JSON文件"按钮
3. 浏览器将下载 `articles.json` 文件，保存到本地

#### 4.2 导入文章数据

1. 点击"导入导出"标签页
2. 点击"选择JSON文件"按钮，选择要导入的 `articles.json` 文件
3. 点击"导入"按钮
4. 系统将使用导入的文件替换当前文章数据

**注意：** 导入操作会覆盖当前所有文章数据，请谨慎操作！

## 四、文章分类

系统支持以下文章分类：

- 基础理论（basic-theory）
- 名人观点（celebrity-views）
- 花果论心法（flower-fruit-method）
- 命理随笔（essays）

## 五、技术实现

- **前端框架**：纯HTML/CSS/JavaScript，无外部框架依赖
- **数据存储**：JSON文件（articles.json）
- **管理页面**：单页面应用，支持响应式设计

## 六、更新文章数据到服务器

1. 使用文章管理页面编辑、添加或删除文章
2. 点击"导出为JSON文件"按钮，下载最新的 `articles.json` 文件
3. 使用FTP工具或网站后台，将下载的 `articles.json` 文件上传到网站根目录，替换原有文件
4. 刷新网站页面，新文章将自动显示

## 七、常见问题

### Q1：为什么我添加的文章没有显示在网站上？
A1：请确保您已经将更新后的 `articles.json` 文件上传到服务器根目录。

### Q2：如何修改管理员密码？
A2：在 `admin.html` 文件中找到 `ADMIN_PASSWORD` 常量，修改为您想要的密码。

### Q3：可以添加新的文章分类吗？
A3：可以，但需要修改 `admin.html` 文件中的分类选项，以及 `index.html` 中对应的文章板块HTML结构。

### Q4：文章内容支持图片吗？
A4：支持，您可以在文章内容中直接插入图片HTML标签，例如：
```html
<img src="image-url" alt="图片描述">
```

## 八、安全注意事项

1. 请定期备份 `articles.json` 文件，防止数据丢失
2. 建议修改默认管理员密码
3. 不要在公共场合使用管理页面
4. 定期检查网站文件，确保没有被恶意修改
