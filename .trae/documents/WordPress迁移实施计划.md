# 文章管理功能实现计划

## 一、计划概述
在现有静态网站基础上，添加文章管理功能，允许用户通过简单的表单上传和管理文章，无需复杂的后台系统。

## 二、实施步骤

### 1. 文章数据分离
- 将当前HTML文件中的`articlesData`变量分离到独立的`articles.json`文件中
- 前端通过`fetch` API从JSON文件加载文章数据

### 2. 创建文章管理页面
- 创建`admin.html`页面，包含文章管理表单
- 实现文章的添加、编辑、删除功能
- 添加表单验证

### 3. 实现JSON文件操作
- 使用JavaScript的`FileReader` API读取本地JSON文件
- 使用`FileSaver.js`库保存修改后的JSON文件
- 提供上传按钮，允许用户将修改后的JSON文件上传到服务器

### 4. 修改前端代码
- 更新`index.html`中的文章加载逻辑，从`articles.json`文件获取数据
- 确保文章分类和显示正常

### 5. 添加权限控制
- 为管理页面添加简单的密码保护
- 防止未授权访问

## 三、用户需要提供的内容

1. **文章内容**：
   - 文章标题、摘要、完整内容
   - 文章分类（基础理论、名人观点、花果论心法）
   - 发布日期

2. **管理密码**：
   - 设置一个简单的密码，用于访问文章管理页面

## 四、使用说明

1. **访问管理页面**：
   - 在浏览器中输入 `https://www.fatenet.cn/admin.html`
   - 输入密码登录

2. **添加文章**：
   - 填写文章表单（标题、摘要、内容、分类、日期）
   - 点击"添加文章"按钮
   - 下载生成的`articles.json`文件

3. **编辑文章**：
   - 选择要编辑的文章
   - 修改文章内容
   - 点击"保存修改"按钮
   - 下载生成的`articles.json`文件

4. **删除文章**：
   - 选择要删除的文章
   - 点击"删除文章"按钮
   - 下载生成的`articles.json`文件

5. **上传文件**：
   - 使用FTP工具或网站后台上传下载的`articles.json`文件到网站根目录
   - 刷新网站页面，新文章将自动显示

## 五、预期成果

- 简单易用的文章管理页面
- 支持添加、编辑、删除文章
- 无需复杂的后台系统
- 与现有网站无缝集成
- 保持网站的原有功能和风格
